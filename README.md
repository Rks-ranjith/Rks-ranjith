<br/>

Autonomous AI-assisted web pentesting platform that orchestrates a **7-stage attack pipeline** via a finite-state-machine architecture, using AI-driven analysis to prioritize and verify findings. Includes a headless Playwright crawler for automated SPA/API endpoint discovery, **19 modular vulnerability-verification plugins** covering OWASP Top 10 attack classes, and a React + TypeScript dashboard streaming real-time WebSocket telemetry with an interactive attack-graph visualization.

| Aspect | Detail |
|---|---|
| **Stack** | Python · React · TypeScript · Playwright · WebSockets |
| **Scale** | 7-stage FSM pipeline · 19 verification plugins |
| **Performance** | Real-time WebSocket telemetry & live attack-graph rendering |
| **Security** | Detects SQLi, XSS, IDOR, Broken Authentication |
| **Impact** | Validated against OWASP Juice Shop & DVWA across multiple endpoints |
| **Repository** | [github.com/Rks-ranjith/AWAP-AI](https://github.com/Rks-ranjith/AWAP-AI) |

Built to prove that offensive tooling can be both automated *and* explainable — the AI layer doesn't just fire payloads, it reasons about which findings matter and verifies them before surfacing to the dashboard.
